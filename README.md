# GenericList

Custom implementation of List<T> in C# built for learning data structures, generics, and collection internals.

## Features
- Add / AddRange
- Insert / InsertRange
- Remove / RemoveAt / RemoveRange
- IndexOf / Contains
- Reverse
- ToArray
- IEnumerable<T> support

## Concepts Practiced
- Generics
- Interfaces
- IEnumerable<T>
- Dynamic Resizing
- Collection Design
